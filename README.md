## 🧭 What clawspark does

clawspark helps you set up a local AI agent on supported NVIDIA hardware with one click. It is built for NVIDIA DGX Spark, Jetson, and RTX systems. It pairs OpenClaw with Ollama so you can run AI tools on your own machine.

This project is made for end users who want a fast setup on Windows. It focuses on local use, simple setup, and direct access to the release download page.

## 💻 What you need

Before you start, check that your PC has:

- Windows 10 or Windows 11
- An NVIDIA GPU or NVIDIA-based device
- Enough free disk space for the app and AI models
- A stable internet connection for the first download
- Admin access if Windows asks for it

For best results, use:

- 16 GB RAM or more
- 20 GB of free storage or more
- Recent NVIDIA drivers
- A machine that can run NVIDIA CUDA apps

## 🛠️ Common setup tips

If the install does not start, try these steps:

- Re-download the file from the release page
- Run the installer as admin
- Check that your Windows version is current
- Update your NVIDIA driver
- Close other heavy apps before setup
- Make sure you have enough free disk space

If the app opens but feels slow:

- Restart the app
- Let the first model finish loading
- Check that your GPU is active
- Make sure Windows is not in power saver mode

## ❓ If the app asks for a model

Some local AI apps need a model file before they can answer questions. If clawspark asks for one, choose the model it suggests or use the default local model path. The app should guide you through the next step.

If you want better speed, use a smaller model. If you want better quality, use a larger model and make sure you have enough RAM and storage.
